# FingerGen_Backend
Django API for Fake Fingerprint Generation

## Run Server

1. Go to FingerGen_Backend

2. 업데이트용

```console
$ python manage.py makemigrations
$ python manage.py migrate
```

3. 실행
```console
$ python manage.py runserver
```

## Conda 환경설정
```console
$ conda env create -f environment.yml
```
